Specify if you want the comment to show in the log
If Yes, before the command line is run, the command will appear. This can be helpful when debugging issues.

Example:

  
/*
Comment about below xshell action:
This is just a test,
Located at https://github.com/ES/testScripts
*/
[workspace] $ cmd.exe /C '"echo Example && exit %%ERRORLEVEL%%"'
Example
Finished: SUCCESS
